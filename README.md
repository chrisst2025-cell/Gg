🔌 API

Minato fonctionne avec une architecture API.

Chat

GET /api/chat

Paramètres :

prompt=
uid=

Exemple :

/api/chat?prompt=Bonjour&uid=123

---

Vision

POST /api/vision

Analyse une image avec l'intelligence artificielle.

---

Image

GET /api/image

Génère une image depuis une description.

---

Mémoire

GET /api/memory

Gestion des souvenirs utilisateur.

2. Installer les dépendances

npm install

3. Démarrer le serveur

npm start

Le projet sera disponible sur :

http://localhost:3000
